# GPU服务器用户监控程序项目

## 项目需求

该项目要实现可按照时间（一天一周还是一个月可选）查看远程GPU服务器上用户的登录情况，和CPU，gpu，内存，显存使用情况。

用户通过QQ机器人输入命令获取上面的信息。

## 技术栈

采用python开发。uv管理项目，uv创建虚拟环境。
