# 🎮 QQ机器人命令设计

> PLAN-004 | v3.0 | 2026-04-07 | status: completed

## 1. 命令体系

### 1.1 命令列表（v3.0 扩展版）

| 命令 | 功能 | 状态 |
|------|------|------|
| `/info` | 查询当前所有在线用户的资源使用情况 | ✅ |
| `/info <用户名>` | 查询指定用户的当前资源使用情况 | ✅ |
| `/info <number>d/w/m` | 查询过去指定天/周/月数的历史记录（精简版） | ✅ |
| `/info <用户名> <时间>` | 查询指定用户的历史记录（精简版） | 📋 待开发 |
| `/stats <时间>` | 显示统计摘要（峰值/平均/使用时长） | 📋 待开发 |
| `/top [gpu/mem/cpu]` | 按资源使用排序的用户排行榜 | 📋 待开发 |
| `/users [时间]` | 显示所有历史用户列表及统计 | 📋 待开发 |
| `/help` | 显示帮助信息 | ✅ |

### 1.2 命令语法

```
@机器人 /info [参数]
```

## 3. 命令解析逻辑

### 3.1 解析规则

```
输入: "/info"
  → action: "info", target: None, time_range: None

输入: "/info user1"
  → action: "info", target: "user1", time_range: None

输入: "/info 7d"
  → action: "info", target: None, time_range: "7d"

输入: "/info 2w"
  → action: "info", target: None, time_range: "2w"

输入: "/info 1m"
  → action: "info", target: None, time_range: "1m"

输入: "/info cxy 3d"
  → action: "info", target: "cxy", time_range: "3d"

输入: "/stats 3d"
  → action: "stats", time_range: "3d"

输入: "/top"
  → action: "top", sort_by: None (默认gpu)

输入: "/top gpu"
  → action: "top", sort_by: "gpu"

输入: "/users"
  → action: "users", time_range: "30d" (默认30天)

输入: "/users 7d"
  → action: "users", time_range: "7d"

输入: "/help"
  → action: "help"
```

### 3.2 时间范围映射

| 输入 | time_range | 说明 |
|------|------------|------|
| `7d` | `7d` | 过去7天 |
| `14d` | `14d` | 过去14天 |
| `2w` | `2w` | 过去2周 |
| `1m` | `1m` | 过去1月 |
| `3m` | `3m` | 过去3月 |

## 4. 数据来源

### 4.1 实时查询

直接执行系统命令获取：
- `nvidia-smi` → GPU使用率、显存
- `free` → 内存使用
- `top` → CPU使用率
- `who` → 在线用户

### 4.2 历史查询

从SQLite数据库查询：
- 表: `resource_history`
- 条件: `timestamp > NOW() - INTERVAL`
- 按用户和时间分组

## 5. 错误处理

| 场景 | 回复消息 |
|------|----------|
| 查询失败 | `❌ 查询失败，请稍后重试` |
| 用户不存在 | `❌ 用户不存在或未在线` |
| 数据库错误 | `❌ 历史记录查询失败` |
| 无历史数据 | `📭 暂无历史记录` |
